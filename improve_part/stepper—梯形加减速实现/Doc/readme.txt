【*】 程序简介 
-工程名称：步进电机--梯形加减速实现
-实验平台: 野火STM32 F103 霸道 开发板



【 ！】功能简介：
实现步进电机-梯形加减速实现


【 ！】实验操作：
下载本程序，复位开发板即可。

【 ！！】注意事项：
步进电机驱动器使用32细分

/*********************************************************************************************/

步进电机驱动器:
	PUL+<--->3V3
	PUL-<--->PC6
	DIR+<--->3V3
	DIR-<--->PE4
	ENA+<--->3V3
	ENA-<--->PC4
	
	
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：8MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB = 72MHz
-APB2 = 72MHz 
-APB1 = 36MHz
